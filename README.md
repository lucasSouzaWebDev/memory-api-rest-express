# API Rest com Express.js

Exemplo de API utilizando o framework Express.js, registrando os dados enviados em mem√≥ria, junto a um schedule job criando 4 registros a cada 5 segundos.

* URL base: http://localhost:3003

## Rotas

 - **GET /produtos** : retorna todos os produtos;
 - **POST /produtos** : Cria um produto;
 - **PUT /produtos/:id** : Atualiza um produto pelo seu identificador;
 - **DELETE /produtos/:id** : Exclui um produto pelo seu identificador;
